Introduction

Prostate cancer screening is one of the many Healthspan reminders that practitioners and patietns find hard to remember. FHIR and the clinical Quality Framework help to make this and other tasks more manageable.

Technical Overview

In this and other planned implementation guides your author seeks to provide a colleciton of common clinical reminders. This guide is an instance of screening for cancer -- prostate cancer in this case. The guide is based on the FHIR Clinical Reasoning Module and the Clinical Quality Language (CQL) specification. The guide is intended to be used in a FHIR server that supports the Clinical Reasoning Module.
